# Iaret — Character Identity & Asset Specifications

## Name & Etymology

**Iaret** (Egyptian: *iꜥrt*) — "the risen one", "she who rears up". In Egyptian mythology, the uraeus is the rearing cobra worn on the pharaoh's crown as a symbol of divine authority and protection. Iaret is the Ouroboros system's uraeus — the guardian serpent who rises to protect, guide, and embody the system's intelligence.

## Integration with Ouroboros CLI

The CLI loads Iaret assets at runtime for avatar display. After extraction to this sub-repo, assets are referenced via submodule path:

```csharp
// Resolve Iaret assets from the iaret submodule
var iaretPath = Path.Combine(AppContext.BaseDirectory, "..", "..", "..", "..", "iaret", "assets");
```

Or via project file linking:

```xml
<ItemGroup>
  <Content Include="..\..\..\..\iaret\assets\avatar\**"
           Link="Assets\Avatar\Iaret\%(RecursiveDir)%(Filename)%(Extension)"
           CopyToOutputDirectory="PreserveNewest" />
</ItemGroup>
```
